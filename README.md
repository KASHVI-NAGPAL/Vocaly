# 🎤 Vocaly – Speech Practice for Kids

Vocaly is an interactive AI-powered web application designed to help kids improve their pronunciation, vocabulary, and spelling in a fun and engaging way.

---

## 🚀 Features

- 🎙️ **Speech Practice**  
  Speak a word and get instant feedback using speech recognition.

- 🎮 **Games Section**  
  Match the correct word from multiple options.

- ❓ **Quiz Section**  
  Choose the correct answer and test your knowledge.

- ⌨️ **Typing Test**  
  Improve spelling by typing the correct word.

- 🎨 **Interactive UI**  
  Kid-friendly design with gradients, bubbles, and animations.

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (Flask)  
- **AI/Voice:** SpeechRecognition (Google API)  
- **Audio Processing:** pydub + ffmpeg
